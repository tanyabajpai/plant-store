# FloraVision Plant Store

Frontend internship assignment built using Next.js and Tailwind CSS.

## Tech Stack

- Next.js
- React
- Tailwind CSS
- React Icons

## Features

- Responsive design
- Reusable components
- Product cards
- Customer reviews
- Modern UI with glassmorphism

## Run Locally

npm install

npm run dev

## Deployment

Deployed using Vercel.
